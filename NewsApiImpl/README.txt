Steps to run the application:

1.	Run as: Spring boot application
2.	open page http://localhost:8080/home once Tomcat server is started on your browser(Chrome or explorer)
3.	Provide the inputs 1) News description
					   2) number of top sources required
					   
4.  Press submit, you will get the top sources of the article which were retrieved using below News api url

	http://newsapi.org/v2/everything?q=newsDescriotion&sortBy=publishedAt&apiKey=9bceadafe5fa48b5bd00293477ccca59
